Short term courses:



1)Autosar: AUTOSAR (AUTomotive Open System ARchitecture) is a standardized software framework for automotive systems. It offers tools for development, configuration, and integration of embedded software in vehicles




2)Automotive Systems: Automotive systems encompass various interconnected components, including engines, transmissions, electrical systems, and onboard computers, crucial for the functioning of vehicles

3)Embedded C: Embedded C is a specialized programming language used for developing software that runs on embedded systems, emphasizing efficiency, real-time performance, and hardware-specific functionalities

4)Python: Python is a high-level, versatile programming language known for its readability, simplicity, extensive libraries, and suitability for various applications, including web development and data analysis

5)Micro-python: MicroPython is a lean implementation of Python optimized for microcontrollers and embedded systems, offering simplicity, efficiency, and ease for IoT and hardware-related projects






Long term course:



We are going to cover all the following modules :

1)C programming: C is a powerful, procedural programming language with high performance, widely used for system software, embedded systems, and application development.

2)Data structures and algorithms: Data structures and algorithms involve organizing and manipulating data effectively, offering optimized solutions for various computational problems, critical in software development.

3)C++ programming: C++ is an object-oriented programming language derived from C, known for its efficiency, versatility, and extensive use in software development and system programming.

4)Python Programming: Python is a high-level, versatile programming language celebrated for its simplicity, readability, and extensive libraries, widely used in web development, data analysis, and automation.

5)Linux OS/Shell scripting: Linux, an open-source OS, utilizes shell scripting to automate tasks. It involves using shell commands and scripting languages for system administration and process automation.

6)Introduction to Embedded Systems: Embedded systems are specialized computing systems within devices. They control specific functions, integrating hardware and software for applications like IoT, automotive, and consumer electronics.

7)Introduction to automotive systems, introduction to Autosar, CAN, CANoe bus: Automotive systems encompass vehicle functions. AUTOSAR standardizes automotive software. CAN (Controller Area Network) enables communication, while CANoe tests and simulates CAN bus networks.

8)Basics of networking(ISO model, TCP/UDP protocol etc.): Networking follows ISO model's layers for data transmission. TCP provides reliable, ordered delivery, while UDP offers faster, connectionless transmission in computer communications.
